# ANN-Model-for-Customer-Churn-Prediction
The main goal of this project is to analyze customer behavior and predict whether a customer is likely to leave the bank (churn) or continue staying with the company.

This project builds an Artificial Neural Network (ANN) using TensorFlow/Keras to predict customer churn based on banking customer data. The workflow includes data preprocessing, label encoding, one-hot encoding, feature scaling, model training, prediction, and performance evaluation using accuracy score, confusion matrix, and classification report.

**Features
Data preprocessing using Pandas & Scikit-learn
Label Encoding and One-Hot Encoding
Feature scaling with StandardScaler
ANN model creation using TensorFlow/Keras
Customer churn prediction
Model evaluation with confusion matrix and accuracy metrics
Visualization using Matplotlib & Seaborn

**Technologies Used
Python
TensorFlow / Keras
NumPy
Pandas
Scikit-learn
Matplotlib
Seaborn
